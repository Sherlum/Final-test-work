Контрольная работа.


Задача:

Написать программу, которая из имеющегося массива строк формирует массив из строк, длина которых меньше либо равна 3 символа. 
Первоначальный массив можно ввести с клавиатуры, либо задать на старте выполнения алгоритма.
При решении не рекомендуется пользоваться коллекциями, лучше обойтись исключительно массивами.

Описание алгоритма решения:

Объявляются два массива: изначальный и вторый такой же длины.
Затем вызывается метод, в котором цикл соответствует длине массива (for).
Внутри цикла вызывается проверка условия (Меньше или равно 3). 
Если проверка прошла успешна, то элемент первого массива заносится в count элемента второго массива.
Переменная count нужна для того, чтобы по очереди переносить значение из первого массива во второй.
После присвоения, переменная count увеличивается на 1 и возвращается к циклу (for), в котором i увеличивается на 1, пока не достигнет последнего элемента массива.

Навигация:

Блок схема находится в папке Flowchart.
Программа находится в папке Task/Program.cs
